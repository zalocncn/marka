# Designjoy.co redesign (UI kit applied)

This is a static landing page redesign using the provided clinical-modern UI kit (tokens + components).

## View locally

- Open `index.html` directly in your browser, or
- Run a local server from this folder:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Files

- `index.html`: page structure + sections
- `styles.css`: UI kit tokens + components + layout
- `script.js`: mobile nav toggle, FAQ accordion, footer year
